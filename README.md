# Password Strength Game

An interactive browser-based game that teaches password security.

## How to Run
Open index.html in any web browser. No installation needed.

## Features
- Easy / Medium / Hard difficulty (5, 10, 15 rules)
- Live crack-time estimator
- Scoring system with leaderboard
- Password security information panels

## Technologies
HTML5 · Tailwind CSS · Vanilla JavaScript

## Files
- index.html — main application
- assets/css/style.css — styling
- assets/js/script.js — game logic
